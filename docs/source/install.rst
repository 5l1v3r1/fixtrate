Installation
=============

With Pip
----------

Installation with Pip is the hotness, simply::

   $ pip install fixation

Installing From Source
------------------------

Fixation is developed on `Github
<https://github.com/helaoban/fixation.git>`_

You can clone the public repository::

    $ git clone git://github.com/helaoban/fixation.git

Or download the `tarball <https://github.com/helaoban/fixation/tarball/master>`_::

    $ curl -OL https://github.com/helaoban/fixation/tarball/master

And from the root directory, simply::

    $ python setup.py install
